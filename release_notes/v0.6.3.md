## Added

### Namespace export (`GET /wiki/{namespace}/export`)

Logged-in users can now download an entire namespace as a ZIP file
directly from the namespace index page.

**ZIP layout:**

```
Main/
  some-page.md
  another-page.rst
  a-wikitext-page.wiki
  some-page/
    attachments/
      image.png
      document.pdf
```

- Source files use the correct extension for the page format:
  `.md` (Markdown), `.rst` (RST), `.wiki` (Wikitext)
- Attachments are nested under `{slug}/attachments/`
- Only the latest version of each page is included
- Missing attachment files on disk are silently skipped
- Requires login; anonymous users are redirected to `/login`
- No new dependencies — uses Python stdlib `zipfile` and `io`

The **Export ZIP** button appears on every namespace index page for
logged-in users.
